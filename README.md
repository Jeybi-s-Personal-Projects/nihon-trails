NihonTrails

NihonTrails is a travel planning web application built with Next.js that helps travelers discover and plan memorable adventures across Japan. Whether you're searching for cultural landmarks, scenic destinations, local transportation hubs, or accommodations, NihonTrails provides a streamlined experience to make trip planning easier and more enjoyable.

Features

- Attraction Discovery – Search and explore popular attractions based on your selected location.
- Hotel Recommendations – Find nearby accommodations to suit your travel needs.
- Transportation Hub Lookup – Locate train stations, airports, and other transport hubs for seamless travel.
- Personalized Travel Planning – Filter destinations and experiences to match your interests and travel style.
- Location-Based Search – Discover points of interest around your chosen destination.
- API Integrations – Leverages external APIs to provide up-to-date travel information and recommendations.

  Tech Stack

- Frontend: Next.js
- Styling: Tailwind CSS
- APIs: Third-party travel, location, and accommodation services
- Deployment: Vercel

  Project Purpose

NihonTrails was created as a personal passion project to combine an interest in travel and web development. It serves as both a practical trip planning tool and an opportunity to strengthen skills in modern web technologies, API integration, and responsive application design.
